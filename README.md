# SQL Portfolio — BigQuery

A collection of SQL queries written on **Google BigQuery** using the public dataset
`bigquery-public-data.thelook_ecommerce`.

## Skills Demonstrated
- SELECT, FROM, WHERE, filtering and logical operators
- Aggregations: COUNT, SUM, AVG, GROUP BY, HAVING
- JOINs: INNER, LEFT, FULL OUTER
- Subqueries (WHERE clause, FROM clause)
- CASE WHEN for segmentation and conditional logic
- String and date functions
- CTEs (Common Table Expressions)
- Window Functions: ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD

## Dataset
`bigquery-public-data.thelook_ecommerce` — a public ecommerce dataset available
on Google BigQuery, simulating an online fashion retail business.

## Tools
- Google BigQuery
- Visual Studio Code
- SQL dialect: BigQuery Standard SQL

## About
7-day structured SQL study plan, progressing from fundamentals to advanced
analytical queries used in real marketing analytics workflows.
